# Política de seguridad

## Reportar una vulnerabilidad

Si encuentras un problema de seguridad en este sitio (doxdesigns.dev) o en su código,
escríbeme de forma privada a **santiago.miranda.trabajo@gmail.com**. Por favor **no** abras
un issue público hasta que el problema esté resuelto.

Incluye, si puedes:

- Una descripción del problema y su posible impacto.
- Pasos para reproducirlo (URL, navegador, capturas).

Intentaré responder en un plazo razonable y te daré crédito si así lo deseas.

## Alcance

Este es un portafolio personal **estático** (sin backend, base de datos ni datos de
usuarios). El interés principal son: configuración de cabeceras/CSP, dependencias y
la cadena de despliegue.
